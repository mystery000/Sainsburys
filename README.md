# Sainsburys

This is a trusted web scraper for extracting products from Sainburys.

# How to setup

## Ubuntu

python3 -m venv venv
source venv/bin/activate

pip install requests
pip install html5lib
pip install beautifulsoup4
pip install selenium
pip install pandas

## Windows

python -m venv venv
venv\Scripts\activate

pip install requests
pip install html5lib
pip install beautifulsoup4
pip install selenium
pip install pandas

# How to run

## Ubuntu

source venv/bin/activate
python3 main.py

## Windows

venv\Scripts\activate
python main.py
